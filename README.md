# Iris Flower Classification Using Machine Learning

## Overview

This project demonstrates a basic classification model using the Iris dataset. The model is trained to classify iris flowers into different species based on flower measurements.

## Objectives

* Load and understand a dataset
* Split data into training and testing sets
* Train a classification model
* Make predictions on unseen data
* Evaluate model performance

## Technologies Used

* Python
* Scikit-learn
* NumPy

## Dataset

The project uses the Iris dataset, which contains:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

Target Classes:

* Setosa
* Versicolor
* Virginica

## Methodology

1. Load the Iris dataset.
2. Split the dataset into training and testing sets.
3. Train a Logistic Regression classifier.
4. Predict flower species using the trained model.
5. Evaluate performance using accuracy and confusion matrix.

## Results

The model successfully classifies iris flowers and achieves high prediction accuracy on the test dataset.

## Learning Outcomes

* Understanding supervised learning
* Data preprocessing and handling
* Train-test data splitting
* Classification model training
* Model evaluation and prediction
